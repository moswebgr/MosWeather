MosWeather
A lightweight weather dashboard built by MosWeb. This tool displays the current time and the days of the week, and it fetches live weather data using the Open-Meteo API. Real-time updates make it ideal for embedding in personal projects or dashboards.

Key features:

Displays the current hour, minute and second.

Lists the days of the week (MO, TU, WE, TH, FR, SA, SU).

Retrieves weather information via the Open-Meteo API, ensuring up-to-date weather conditions.

Clean and simple UI designed for easy integration and minimal setup.
